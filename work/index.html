Základy pro založení webu

HTML

Příklad kódu

<!DOCTYPE html>
  <head>
    <title>Název stránky který se zobrazí v kartě stránky</title>
  </head>
  <body>
    <p> Nějaký text </p>
  </body>
  </html>

Tipy

  1. Správná struktura - Vytvořte jasnou strukturu svého HTML dokumentu pomocí prvků jako head, header, main, article, footer.
  2. Semantické prvky - Používejte semantické prvky, jako například h1-h6 pro nadpisy, p pro odstavce, a pro odkazy atd.
  3. Atributy - Používejte atributy, jako například class nebo id, abyste mohli lépe cílit na prvky pomocí CSS.
  4. Odkazy - Používejte odkazy k propojení svých stránek nebo ke zdrojům externích stránek.
  5. Obrázky - Používejte obrázky a grafiku k vylepšení vzhledu a zvýšení atraktivity vašeho webu.
  6. Tabulky - Použijte tabulky k uspořádání dat nebo k vytvoření rozvržení stránky.
  7. Validace - Ověřte svůj kód pomocí nástrojů pro validaci HTML, abyste se ujistili, že je správně napsán.
