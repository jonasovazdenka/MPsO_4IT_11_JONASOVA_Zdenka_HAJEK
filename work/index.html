Základy pro založení webu

HTML

Tipy pro začátečníky

  1. Správná struktura HTML - HTML tag <html>, head tag <head>, body tag <body>
  2. Nezapomeňte uzavírat tagy - <p> Toto je příklad </p>
     POZOR! Některé tagy se uzavárají jinak např. <br/> a <img ... />
  3. Pomocné weby - W3Schools
  4. Stylování stránky - vytvořte nový soubor ".css" nepište to do HTML
     (pojmenujte ho jak chcete, nejčasteji je používání style.css, tak vám to i html navrhne při linkování)
     pak stačí soubor pouze "nalinkovat": <link rel="stylesheet" href="style.css">
  5. Správné značky pro nadpisy - První/největší nadpis bude mít tag <h1> Podnadpis bude mít tag <h2> apod. (max. tag je <h6>)
      Pomáhá to se lépe zorientovat na stránce.
  6. Alt tag - používá se třeba při img tagu (obrázek). Když uživatel najede kurzorem na obrázek, zobrazí se text který je napsaný v alt tagu.

JavaScript

Tipy

  1. Naučte se základní syntaxi a datové typy (proměnné, funkce, objekty, pole).
  2. Pracujte s DOM (Document Object Model) pro manipulaci s HTML elementy.
  3. Využívejte knihovny a frameworky jako jQuery nebo React pro vývoj aplikací.
  4. Používejte konzolu pro debugování a ladění kódu.
  5. Sledujte online tutoriály a cvičení, abyste se učili od odborníků.
  6. Pracujte na projektech a vystavujte je na GitHubu, abyste zlepšili své portfolio.
  7. Sdílejte své projekty a kód s jinými vývojáři a získávejte zpětnou vazbu.
  8. Učte se nové funkce a API.
  9. Přečtěte si dokumentaci k různým knihovnám a frameworkům.
  10. Začněte s jednoduchými úlohami jako například kalkulačka nebo to-do aplikace.
